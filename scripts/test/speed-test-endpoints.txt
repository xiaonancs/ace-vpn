# ace-vpn · VPS 出站质量长期测速端点（每行一个 URL）。
# vps-watch-urls.sh 默认会合并本文件 + private/vps-watch-urls.txt（去重）。
# 只保留常见海外服务，主要用于判断 VPS 到美国头部互联网公司产品的出站质量。
# 控制在 100 个 URL 以内，避免每 30 分钟一轮的探测时间过长。
